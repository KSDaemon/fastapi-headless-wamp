## Codebase Patterns
- Package manager is `uv`; use `uv run` to execute commands in the venv
- Source layout: `src/fastapi_headless_wamp/` with hatchling build backend
- Typecheck: `uv run pyright src/` (strict mode, Python 3.12 target)
- Tests: `uv run pytest` with pytest-asyncio in auto mode
- README.md is required by hatchling build (even if minimal)
- pyright config is in pyproject.toml under `[tool.pyright]`
- In pyright strict mode, use typed factory functions for dataclass default_factory (e.g., `def _dict_factory() -> dict[str, Any]: return {}`) instead of bare `dict`/`list`
- Message dataclasses use deferred imports inside `to_list()` methods to avoid circular imports between types.py and protocol.py
- `validate_message()` takes `Any` (not `list[Any]`) so the isinstance check is meaningful at runtime
